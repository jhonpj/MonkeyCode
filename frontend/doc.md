# MonkeyCode AI 前端使用文档

本文档基于当前 `frontend` 源码整理，描述的是现在前端已经实现的页面、入口和交互。

如果文档与页面行为不一致，优先以以下源码为准：

- 路由入口：`frontend/src/App.tsx`
- 页面组件：`frontend/src/pages/**`
- 关键交互组件：`frontend/src/components/**`

---

## 一、产品与页面入口

### 1. 当前前端覆盖的能力

当前前端已经覆盖以下能力：

- 官网首页与公开介绍页
- 普通用户登录与控制台
- 智能任务创建、执行、追问、上下文管理、文件查看和在线预览
- 项目协作、需求管理和自动 Review
- Git 审查机器人
- 独立开发环境、终端、文件管理、端口管理和远程协助
- 开发者广场内容浏览与发布
- 企业管理后台

### 2. 主要路由

- 首页：`/`
- 登录页：`/login`
- 找回密码：`/findpassword`
- 重置密码：`/resetpassword?token=...`
- 开发者广场：`/playground`
- 发布广场内容：`/playground/create`
- 广场详情：`/playground/detail?id=...`
- 隐私政策：`/privacy-policy`
- 用户协议：`/user-agreement`
- 产品介绍 PPT：`/ppt/` 或 `/ppt/index.html`
- 公开任务跳转入口：`/tasks/public?id=...`
- 普通用户控制台：`/console`
- 新任务页：`/console/tasks`
- 任务详情页：`/console/task/:taskId`
- 项目页：`/console/project/:projectId`
- Git 机器人：`/console/gitbot`
- IDE 辅助工具：`/console/ide`
- 独立终端页：`/console/terminal?envid=...`
- 独立文件管理页：`/console/files?envid=...&path=...`
- 远程协助页：`/sharedterminal?id=...`
- 企业管理后台：`/manager`

### 3. 首页导航

首页顶部导航当前包含：

- `介绍`
- `广场`
- `文档` / `使用文档`
- `开源` / GitHub 入口
- 已登录时显示 `控制台` / `进入控制台`
- 未登录时显示 `注册`、`立即开始`

## 二、登录与账号

### 1. 登录页

登录页路径：

- `/login`

页面包含两个标签：

- `普通用户`
- `团队管理员`

登录页底部要求勾选并同意：

- `用户协议`

如果未勾选协议，普通用户 OAuth 登录、快速注册、账号密码登录和团队管理员登录都会被阻止。

### 2. 普通用户登录

普通用户标签默认先展示登录方式选择：

- `百智云登录 - 推荐`
- `账号密码登录`
- `快速注册`

说明：

- `百智云登录 - 推荐` 和 `快速注册` 都跳转到后端登录入口 `/api/v1/users/login?redirect=&inviter_id=...`
- 如果首页或注册入口保存过邀请码，登录入口会携带 `inviter_id`
- `账号密码登录` 会切换到邮箱 / 密码表单

账号密码登录支持：

- 输入邮箱和密码
- 密码明文 / 隐藏切换
- 登录前验证码校验
- 本地记住上次输入的邮箱和密码
- `找回密码`

普通用户登录成功后进入：

- `/console/`

### 3. 团队管理员登录

团队管理员登录表单支持：

- 输入邮箱和密码
- 密码明文 / 隐藏切换
- 登录前验证码校验
- 本地记住上次输入的邮箱和密码

团队管理员登录成功后进入：

- `/manager/`

团队管理员标签下当前没有单独的 `找回密码` 入口。

### 4. 找回密码

找回密码页路径：

- `/findpassword`

操作流程：

1. 输入邮箱
2. 完成验证码校验
3. 提交后发送重置密码邮件

### 5. 重置密码

重置密码页路径：

- `/resetpassword?token=...`

页面会先根据 `token` 读取账号信息，再要求输入：

- 新密码
- 确认密码

前端校验：

- 两次密码必须一致
- 密码长度至少 6 位

重置成功后会提示用户，并返回首页。

## 三、普通用户控制台

### 1. 默认入口

登录后访问 `/console` 会自动跳转到：

- `/console/tasks`

控制台布局当前为：

- 左侧侧边栏
- 顶部面包屑与操作区
- 主内容区

顶部区域包含：

- 面包屑
- 免费模型用量提示
- `获取积分`
- `刷新`

### 2. 免费模型用量提示

顶部免费模型提示会展示平台共享免费模型池的当日用量。

当前实现：

- 每天早上 10 点送出 500 亿 token 免费额度
- 所有用户共享使用
- 选择带有 `免费` 标记的模型即可使用免费额度
- 当日免费额度耗尽后，模型调用按正常模型价格扣除积分
- 前端每 30 秒请求一次 `/api/v1/users/free-model-pool/usage`

### 3. 获取积分

顶部 `获取积分` 会打开弹窗，当前列出：

- 邀请注册：`+5,000 积分 / 人`
- 征文投稿：`1 万 - 10 万积分`
- 充值积分：立即到账
- 每日签到：`+100 积分 / 天`
- 社区活动：不定期奖励

其中：

- `去邀请好友` 和 `去充值` 会打开钱包的 `我的积分`
- `加入社区交流群` 会打开技术交流群弹窗
- 征文投稿会跳转到外部活动规则文档

### 4. 左侧侧边栏

侧边栏包含：

- 顶部 MonkeyCode 标识
- 未归属任务和项目列表
- `签到领积分`
- `技术交流群`
- `配置`
- 当前用户与积分入口

任务与项目区域支持：

- 查看未归属任务
- 点击 `启动任务` 打开快速任务创建弹窗
- 展开 / 折叠 `历史任务`
- 点击 `查看更多` 进入 `/console/tasks`
- 查看项目列表
- 点击项目进入项目页
- 对项目点击 `+` 直接启动项目任务
- 点击 `添加项目` 创建项目
- 每 30 秒刷新项目、未归属任务和历史任务

任务条目支持：

- 进入任务详情
- 修改任务名称
- 对等待中 / 运行中的任务执行 `终止任务`
- 删除任务

折叠侧边栏时：

- 显示 `空项目`
- 显示 `项目列表` 或 `创建项目`

### 5. 创建项目

创建项目当前不是自由输入仓库地址，而是：

1. 输入项目名称
2. 从已绑定 Git 身份中选择身份
3. 从该身份可访问的授权仓库列表里选择仓库

如果尚未绑定 Git 身份，弹窗会提示先去 `配置` 里完成绑定。

### 6. 技术交流群

侧边栏 `技术交流群` 和顶部 `获取积分` 弹窗都可以打开社群入口。

当前展示：

- 微信群二维码
- 飞书群二维码
- 钉钉群二维码

### 7. 签到

如果当天尚未签到，侧边栏会显示 `签到领积分`。

当前实现：

- 点击后先进行验证码校验
- 签到成功获得 100 积分
- 成功后刷新钱包余额和签到状态
- 如果当天已签到，侧边栏签到入口不显示

## 四、钱包、套餐与积分

### 1. 钱包入口

侧边栏底部用户入口同时展示：

- 用户头像和昵称
- 当前套餐简称
- 当前可用积分总数

当前可用积分 = `长期积分` + `今日积分`。

点开后，钱包弹窗包含 5 个分区：

- `我的账户`
- `我的套餐`
- `我的积分`
- `积分记录`
- `模型定价`

### 2. 我的账户

`我的账户` 主要展示：

- 用户头像、昵称、邮箱
- 当前套餐
- 可用积分
- 所属团队或个人空间

账号操作包括：

- 修改头像
- 修改昵称
- 绑定邮箱
- 修改密码 / 设置密码
- 登出

补充说明：

- 头像必须是图片文件
- 已有密码的账号修改密码时需要输入当前密码
- 设置 / 修改用户密码时，前端要求新密码至少 8 位
- 绑定邮箱会发送绑定邮件

### 3. 我的套餐

套餐页当前展示：

- `基础版`
- `专业版`
- `旗舰版`

基础版：

- 免费
- 并发任务限制：1 个任务
- 每日积分：无

专业版：

- 价格：10,000 积分 / 月
- 并发任务限制：3 个任务
- 每日赠送 2,000 积分
- 每日积分仅当日有效，不累计

旗舰版：

- 价格：100,000 积分 / 月
- 并发任务限制：3 个任务
- 每日赠送 30,000 积分
- 每日积分仅当日有效，不累计

套餐操作：

- 非团队用户且已开通高级套餐时，顶部显示到期时间和自动续费开关
- 可直接开通专业版或旗舰版
- 开通前会弹出确认框
- 开通成功后刷新钱包、套餐状态和交易记录
- 如果当前是旗舰版，不再显示专业版开通按钮

### 4. 我的积分

`我的积分` 当前包含：

- 当前可用积分
- 邀请注册
- 社区活动
- 每日签到
- 兑换积分

当前可用积分拆分为：

- `长期积分`
- `今日积分（凌晨自动清零）`

充值：

- 点击 `充值` 打开充值弹窗
- 选择充值档位后点击 `确认充值`
- 前端向后端请求支付链接
- 请求成功后在新标签页打开支付页面

固定充值档位：

- `10,000 积分`，价格 `¥50`
- `50,000 积分`，价格 `¥200`，原价 `¥250`，页面标记为 `8 折`
- `300,000 积分`，价格 `¥1000`，原价 `¥1500`，页面标记为 `6.7 折`

- 系统生成个人邀请链接：`https://monkeycode-ai.com/?ic=当前用户ID`

每日签到：

- 每天可签到 1 次
- 每次获得 100 积分
- 钱包内也可发起签到

兑换积分：

- 输入兑换码后点击 `兑换`

## 五、智能任务

### 1. 新任务页

新任务页路径：

- `/console/tasks`

页面由两部分组成：

- 上方任务输入区
- 下方任务卡片列表

页面标题：

- `MonkeyCode 智能任务`

### 2. 创建任务

#### 任务输入

主输入框支持：

- 输入自然语言需求
- 语音输入

当前输入框没有在代码里单独处理 `Enter` 提交，主要通过点击 `执行` 创建任务。

#### 任务类型

当前有三种类型：

- `开发`
- `设计`
- `审查`

说明：

- `开发`：根据需求执行开发编码任务
- `设计`：进行架构设计，输出技术方案与设计文档
- `审查`：审查代码，识别风险，提出改进建议

#### 代码来源

`代码` 菜单当前支持三种来源：

- `ZIP 文件`
- `我的仓库`
- `其他仓库`

`ZIP 文件`：

- 仅接受 `.zip`
- 文件大小上限 10MB
- 先上传到对象存储，再作为本次任务代码来源

`我的仓库`：

- 按已绑定 Git 身份分组展示
- 支持 GitHub App、Gitee、Gitea、GitLab OAuth 身份
- 支持在每个身份下搜索仓库
- 支持刷新单个身份下的授权仓库列表
- 选中仓库后自动关联对应 Git 身份凭证
- 如果没有可选 Git 身份，会提示去设置绑定

`其他仓库`：

- 手动输入完整仓库地址
- 按回车确认
- 下方展示从任务列表中归纳出的历史仓库地址
- 如果手动输入的仓库地址匹配已绑定身份的 `base_url`，参数确认时可选择对应身份；否则可匿名

#### 技能

`技能` 按钮会打开技能选择面板。

当前实现：

### 8. 文件面板

- 刷新文件树
- 在当前目录创建文件
- 创建目录

- 复制文件
- 移动文件 / 目录
- 下载文件 / 目录
- 删除文件 / 目录

#### 变更模式

- 显示新增 / 删除行数

### 9. 终端面板

点击顶部 `终端` 后，会在页面下方展开终端区。

终端区支持：

- 关闭终端会话

### 10. 在线预览

弹窗支持：

## 十、独立终端、文件管理与远程协助

### 1. 独立终端

路径：

- `/console/terminal?envid=...`

这是针对某个开发环境单独打开的终端页。

当前能力包括：

- 查看开发环境名称、状态、操作系统
- 首次进入时选择已有终端连接
- 新建终端连接
- 连接已有终端连接
- 关闭终端连接
- 手动重新连接
- 切换终端配色主题
- 打开独立文件管理
- 打开端口管理 / 在线预览
- 发起远程协助

终端连接选择弹窗中会显示：

- 终端连接 ID
- 创建时间
- 当前连接数

### 2. 端口管理

独立终端页里的 `在线预览` 实际打开端口管理弹窗。

当前支持：

- 查看端口列表
- 刷新端口列表
- 开放访问
- 访问已开放端口
- 复制访问地址
- 编辑白名单 IP
- 关闭访问 / 回收端口

补充说明：

- 开放访问时，前端会先尝试获取当前访问者 IP
- 如果获取成功，会把该 IP 写入白名单
- 白名单编辑弹窗要求每行输入一个 IP，且至少填写一个 IP

### 3. 远程协助

独立终端页支持生成远程协助连接信息。

模式有两种：

- `只读模式`：对方只能看到终端界面，无法输入或操作
- `控制模式`：对方可以看到终端界面，且可进行输入和交互

生成后会得到：

- 共享地址
- 连接密码

支持：

- 一键复制整段邀请信息

### 4. 远程协助页

路径：

- `/sharedterminal?id=...`

访问者进入后需要先输入连接密码，之后才能加入共享终端。

页面支持：

- 查看连接状态
- 查看分享者名称和头像
- 切换终端主题

### 5. 独立文件管理

路径：

- `/console/files?envid=...&path=...`

这是针对某个开发环境的完整文件管理页。

当前支持：

- 浏览目录
- 面包屑导航
- 返回上级目录
- 刷新文件列表
- 新建文件
- 新建目录
- 上传文件
- 复制文件
- 移动文件 / 目录
- 下载文件 / 目录
- 在线编辑文本文件
- 删除文件

列表展示字段包括：

- 名称
- 大小
- 用户
- 权限属性
- 修改时间

补充说明：

- 超过 1MB 的文件不允许在线编辑
- 当前独立文件管理页里，复制和删除菜单只对文件开放
- 移动和下载支持文件 / 目录

## 十一、IDE 辅助工具

路径：

- `/console/ide`

当前这个页面仍是占位页，但已经提供两个外链：

- 开源仓库
- 官方文档

页面文案为 `敬请期待`，不应描述为已完整交付的功能页。

## 十二、开发者广场

### 1. 广场列表

路径：

- `/playground`

当前已实现：

- 搜索内容
- 浏览文章 / 任务分享卡片
- 查看封面、标题、摘要
- 查看作者与浏览量
- 登录后发布内容
- 未登录点击 `发布` 会跳到登录页

### 2. 发布内容

路径：

- `/playground/create`

当前可选发布类型只有两种：

- `写一篇文章`
- `分享你执行过的任务`

说明：

- 代码里仍保留了 `project` 类型的旧状态变量，但当前 UI 没有开放项目分享入口
- 如果以 `/playground/create?taskid=...` 打开，页面会自动进入任务分享模式并预选该任务

#### 文章发布

支持：

- 标题
- Markdown 内容
- 图片上传

#### 任务分享

支持：

- 从自己的任务列表中选择任务
- 填写标题
- 填写 Markdown 内容
- 上传图片

补充说明：

- 当前前端要求至少上传 1 张图片才能发布
- 当所选任务仍在运行中，且开发环境在线时，还可以额外选择 `产出物` 文件
- 这些产出物会先被前端打包成 ZIP，再随分享一起上传
- 发布成功后提示 `审核后会显示在广场上`

### 3. 广场详情

路径：

- `/playground/detail?id=...`

广场详情页会根据内容类型展示：

- 标题
- 作者
- 发布时间
- 浏览量
- Markdown 正文
- 图片预览

对于任务分享，当前还额外支持：

- 封面上点击播放后，加载并回放任务历史消息
- 下载分享时附带的 ZIP 代码产出物
- 文件树展开 / 折叠
- 打开单个文件查看内容

补充说明：

- 页面右侧 `网友评论` 区域目前只是占位，没有真正评论交互

## 十三、公开任务

路径：

- `/tasks/public?id=...`

这个页面当前不是独立展示页，而是一个跳转入口：

- 如果带有 `id`，会直接跳转到 `/console/task/:taskId`
- 如果没有 `id`，会跳回首页

因此不应把它描述成公开任务列表页或独立公开详情页。

## 十四、企业管理后台

### 1. 路径与导航

企业管理后台路径：

- `/manager`

默认进入：

- `/manager/dashboard`

左侧菜单当前包含：

- 仪表盘
- 成员管理
- 开发环境
- AI 大模型
- 系统镜像
- 管理员
- 操作记录
- 其他配置

右上角提供：

- `刷新`

左下角管理员用户菜单支持：

- 修改密码
- 登出

### 2. 仪表盘

当前仍是占位页，显示：

- `敬请期待`

### 3. 成员管理

成员管理页由两张卡片组成：

- `团队成员`
- `分组`

#### 团队成员

支持：

- 查看成员数量与上限
- 批量添加成员
- 添加时一次输入多个邮箱
- 添加成员时支持逗号、换行或分号分隔邮箱，并会校验邮箱格式
- 禁用 / 启用成员
- 为成员发送重置密码邮件
- 查看成员加入时间和最近使用时间
- 查看成员所属分组

#### 分组

支持：

- 新建分组
- 修改分组名称
- 删除分组
- 查看分组成员数量
- 查看分组成员列表
- 调整分组成员

### 4. 开发环境

团队级开发环境页面管理的是宿主机。

支持：

- 绑定宿主机
- 查看宿主机状态、规格、架构和 IP
- 查看宿主机上的开发环境数量
- 修改宿主机信息
- 删除宿主机
- 查看宿主机所属分组
- 查看宿主机上正在使用的虚拟机

修改宿主机时可配置备注和可使用该宿主机的分组。

### 5. AI 大模型

团队模型支持：

- 添加
- 修改
- 健康检查
- 删除
- 查看所属分组

添加 / 修改团队模型时可配置接口格式、模型 API 地址、API Token、模型名称和可使用该模型的分组；保存前会执行模型健康检查。

### 6. 系统镜像

团队镜像支持：

- 添加
- 修改
- 删除
- 查看所属分组

添加 / 修改团队镜像时可配置镜像名、备注和可使用该镜像的分组。

### 7. 管理员

管理员页支持：

- 查看所有管理员
- 添加管理员
- 为管理员发送密码重置邮件

### 8. 操作记录

操作记录页支持：

- 查看审计日志表格
- 查看时间、用户、操作、来源 IP
- 查看请求内容
- 查看响应内容
- 游标分页
- 切换每页条数

### 9. 其他配置

当前 `其他配置` 里已经实现的是团队级消息通知。

能力与用户侧通知基本一致，支持：

- 钉钉机器人
- 飞书机器人
- 企业微信机器人
- Webhook
- 添加
- 编辑
- 测试
- 删除
- 选择订阅事件
