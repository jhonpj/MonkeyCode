10FAQ

围绕免费、数据、模型和生产使用建立信任

Q01
真的完全免费？怎么赚钱？
个人 Free tier 长期可用。平台主要通过 Pro 订阅和企业自托管商业支持赚钱。
Q02
代码会不会被拿去训练模型？
默认不会。仓库、prompt 和输出默认不进入模型训练流程。自托管版本数据也不会出内网。
Q03
支持哪些模型？
已接入 GPT、Claude、GLM、Kimi、MiniMax、Qwen、DeepSeek 等，也支持第三方兼容接口。
Q04
能用在生产项目上吗？
可以。所有修改都可以回到 Git PR 流程，团队保留 review、审计和 rollback 能力。
